# Portfolio Project

Full-stack portfolio application with TypeScript, Node.js, Express, React, Tailwind CSS, and MongoDB.

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Requirements
- Node.js 18+
- MongoDB running on localhost:27017

## Ports
- Backend: http://localhost:5000
- Frontend: http://localhost:3000
